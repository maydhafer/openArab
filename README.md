وش هو OpenClaw؟🦞

تخيل إنك عندك **مساعد شخصي بالذكاء الاصطناعي**  

يشتغل على جهازك أنت،  وتتحكم فيه بالكامل. هذا المساعد يقدر: - يرد عليك على **واتساب، تيليغرام، ديسكورد، سلاك، آي مسج** وغيرها كلها في مكان واحد
**يسمع صوتك ويكلمك** — مو بس نصوص - **يفتح المتصفح ويتصفح** بدالك ويسوي مهام تلقائية - **يتذكر** تفضيلاتك وأسلوبك ويتعلم منك مع الوقت
يشتغل **24 ساعة** بدون ما تفتح جهازك (لو حطيته على سيرفر) 

💰 وش الفايدة العملية؟ وكيف تربح منه؟
كثير ناس ما يعرفون وش يسوون بمساعد ذكاء اصطناعي — هنا أمثلة حقيقية:
**للأفراد:**
- رد تلقائي على رسائل واتساب وأنت نايم
مساعد يحجز مواعيدك ويذكّرك بمهامك - يبحث لك على الإنترنت ويلخص لك النتائج - يكتب لك محتوى سوشيال ميديا بأسلوبك 
**للأعمال والمشاريع:**
- بوت خدمة عملاء على واتساب يرد 24/7 بدون موظف - مساعد مبيعات يرد على استفسارات العملاء فوراً
-  أتمتة المهام المتكررة (إرسال تقارير، متابعة طلبات، إلخ) - توفير آلاف الريالات شهرياً بدل اشتراكات متعددة **والأهم: كل هذا مجاناً**
البرنامج مفتوح المصدر، تدفع بس لنموذج الذكاء الاصطناعي اللي تختاره (Claude أو ChatGPT)، وأسعارهم رخيصة جداً مقارنة بالاشتراكات الجاهزة

<h2>🚀 طريقة التثبيت — خطوة بخطوة</h2>

<blockquote>
قبل ما تبدأ، تأكد إن جهازك جاهز. المتطلب الوحيد هو <strong>Node.js نسخة 22 أو أحدث</strong>. إذا ما عندك ثبّته أولاً من <a href="https://nodejs.org">nodejs.org</a> ثم ارجع هنا.
</blockquote>

<h3>⚙️ الطريقة الأولى: التثبيت السريع (الموصى بها للمبتدئين)</h3>

هذي أسهل طريقة وتشتغل على macOS وLinux وWindows (عبر WSL2).

**الخطوة 1 — ثبّت OpenClaw بأمر واحد:**
```bash
npm install -g openclaw@latest
```

> إذا تستخدم pnpm، الأمر يكون: `pnpm add -g openclaw@latest`

**الخطوة 2 — شغّل معالج الإعداد التلقائي:**
```bash
openclaw onboard --install-daemon
```

هذا الأمر بيفتح لك معالج (Wizard) يأخذك خطوة بخطوة عشان تربط حسابك، تختار النموذج، وتفعّل القنوات اللي تبيها (واتساب، تيليغرام، ديسكورد وغيرها). البرنامج كمان بيثبت نفسه كخدمة في الخلفية تشتغل تلقائياً كل ما تفتح جهازك.

**الخطوة 3 — شغّل البوابة:**
```bash
openclaw gateway --port 18789 --verbose
```

بعد ما تشغّل هذا الأمر، افتح متصفحك وروح على: `http://localhost:18789` — من هنا تتحكم بكل شيء.

---

<h3>🛠️ الطريقة الثانية: التثبيت من المصدر (للمطورين)</h3>

إذا تبي تشتغل على الكود مباشرة أو تطور فوقه، اتبع هذي الخطوات:

**الخطوة 1 — حمّل الكود:**
```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
```

**الخطوة 2 — ثبّت الاعتماديات:**
```bash
pnpm install
pnpm ui:build
pnpm build
```

**الخطوة 3 — شغّل معالج الإعداد:**
```bash
pnpm openclaw onboard --install-daemon
```

**الخطوة 4 — شغّل البوابة في وضع المطور (مع إعادة تحميل تلقائي):**
```bash
pnpm gateway:watch
```

<h2>🌐 وش يميز openclaw.ai عن غيره؟</h2>

الموقع الرسمي <a href="https://openclaw.ai">openclaw.ai</a> مو بس صفحة تعريفية، هو بوابتك لفهم المشروع بالكامل. من هناك تلاقي:

<ul>
<li>عرض مباشر لكيف يشتغل المساعد على القنوات المختلفة</li>
<li>معرض أمثلة حقيقية (Showcase) لناس يستخدمونه فعلاً</li>
<li>رابط لمعالج الإعداد التفاعلي (Wizard) اللي يرشدك خطوة بخطوة</li>
<li>تفاصيل عن الروح اللي صُمّم عليها المشروع في صفحة <a href="https://soul.md">soul.md</a></li>
<li>قسم الأسئلة الشائعة FAQ لو واجهتك أي مشكلة</li>
</ul>

<h2>💳 متى تدفع فلوس فعلاً؟</h2>

OpenClaw نفسه مجاني (مفتوح المصدر).

لكن تدفع في الحالات التالية فقط:

<ul>
<li>إذا اخترت نموذج ذكاء اصطناعي مدفوع (مثل مفاتيح API لـ OpenAI أو Anthropic)</li>
<li>إذا استخدمت مهارة تتطلب خدمة خارجية مدفوعة</li>
<li>إذا شغّلته على VPS مدفوع</li>
<li>إذا احتجت أدوات SaaS مرتبطة بحسابات مدفوعة</li>
</ul>
