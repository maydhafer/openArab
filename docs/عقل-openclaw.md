# 🧠 الدخول إلى عقل OpenClaw
### دليل مبسط بالعربي — لكل من يريد التحكم الكامل في وكيله الذكي بدون خلفية تقنية

---

## 📁 أين يسكن العقل؟

كل ذكاء OpenClaw موجود في **7 ملفات نصية** داخل مجلد واحد على السيرفر:
~/.openclaw/workspace/


كل ملف يعلّم الوكيل شيئاً مختلفاً — عدّلها وتحكم في كل شيء.

---

## 🗂️ الملفات السبعة — وش تسوي كل وحدة؟

---

### 🌱 `SOUL.md` — الروح والشخصية

> هذا الملف هو **شخصية الوكيل** — من هو؟ كيف يتكلم؟ ما قيمه؟

**تكتبين فيه مثلاً:**
- اسم الوكيل وطريقة كلامه
- هل يكون رسمي أو ودود؟
- ما الأشياء اللي يركز عليها؟

**مثال:**
```md

أنت مساعد ذكي اسمك "رايق".

تتكلم بالعربي الفصيح البسيط.

تهتم بالتقنية والإنتاجية.

ردودك مختصرة ومفيدة دائماً.
🪪 IDENTITY.md — الهوية والصلاحيات
هذا الملف يحدد من يملك الوكيل وكيف يتعرف عليك.

يحتوي على:

Gateway Token — مفتاح الدخول للسيرفر
Server Token — رمز التحقق من الهوية
⚠️ لا تشارك هذا الملف مع أحد — فيه مفاتيحك السرية.

🤖 AGENTS.md — إعداد الوكلاء
هنا تحدد كيف يفكر الوكيل — أي نموذج ذكاء اصطناعي يستخدم؟

الإعدادات الأساسية:

undefined
الوكيل الرئيسي
النموذج: claude-3-5-sonnet
الذاكرة: مفعّلة
الوكلاء الفرعيين: مفعّلون

**النماذج المتاحة:**
| النموذج | الشركة | الاستخدام |
|---------|--------|-----------|
| `gpt-4o` | OpenAI | سريع وذكي |
| `claude-3-5-sonnet` | Anthropic | تحليل عميق |
| `gemini-pro` | Google | مجاني |

---

### 🚀 `BOOTSTRAP.md` — التشغيل والإعداد الأولي

> هذا الملف يُشغَّل **مرة واحدة فقط** عند بدء الوكيل لأول مرة.

يحتوي على:
- إعدادات البيئة الأولية
- المتغيرات المطلوبة للتشغيل

**المتغيرات الأساسية:**
```bash

OPENCLAW_GATEWAY_TOKEN=مفتاحك_هنا

ANTHROPIC_API_KEY=مفتاح_كلود_هنا

OPENAI_API_KEY=مفتاح_openai_هنا
💓 HEARTBEAT.md — نبض السيرفر
هذا الملف يراقب صحة الوكيل — هل يعمل؟ هل في مشكلة؟

أوامر مفيدة للمراقبة:

undefined
شوف الحاويات الشغّالة
docker ps --format "table {{.Names}}\t{{.Status}}"

شوف آخر سجلات الوكيل
docker logs --tail 50 openclaw

أعد تشغيل الوكيل
docker restart openclaw

### 🛠️ `TOOLS.md` — الأدوات والمهارات

> هنا تضيف **قدرات جديدة** للوكيل — بحث، تقويم، يوتيوب، وأكثر.

**الأدوات المتاحة:**

| الأداة | الوصف |
|--------|-------|
| 🔍 Brave Search | بحث في الإنترنت |
| 📅 Google Calendar | إدارة المواعيد |
| 🎥 YouTube | تلخيص الفيديوهات |
| 🎙️ Whisper | تحويل الصوت لنص |
| 🌐 Perplexity | بحث ذكي بالذكاء الاصطناعي |
| ⚙️ n8n | أتمتة المهام |

**لتفعيل أداة:**
```md
الأدوات المفعّلة
brave-search: مفعّل
google-calendar: مفعّل
whisper: مفعّل

## ⚡ كيف تغذّين عقل الوكيل؟

بعد تعديل أي ملف، أرسلي للوكيل مباشرة:
/reload


أو أعيدي تشغيل الوكيل:

```bash

docker restart openclaw
